Примитивная база данных

Проект представляет собой консольное приложение на Python, реализующее простую модель базы данных.
Программа позволяет создавать и удалять таблицы, добавлять, просматривать, изменять и удалять записи.

Проект собран с использованием Poetry. Для автоматизации команд используется Makefile.
Контроль качества кода осуществляется с помощью линтера Ruff.

Запуск программы выполняется командой poetry run project.

Программа поддерживает команды создания таблиц, работы с данными, получения справочной информации и выхода из приложения.
Ошибки пользовательского ввода обрабатываются автоматически.
Опасные операции (удаление таблиц и записей) требуют подтверждения пользователя.
